About

Astronomy Picture of the Day (APOD) is a website provided by NSA and Michigan Technology University (MTU). The website displays a different image or video of our universe along with a summary written by a professional astronomer.

The APOD website is one of the most popular websites across all federal agencies. The data used is sourced from the NASA Open API portal whose objective is to make NASA data, including imagery, available to application developers to use in their applications and services.

The NASA Daily Picture website uses the APOD API to source the imagery including the title and description. The website allows users to navigate through previous daily pictures published on the APOD website.
